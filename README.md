# Convençoes
## Gerais
- **Código** devem ser escrito em **português**.
- **Commits** devem ser escritos em **português**.
## Variáveis
- Nome das **variáveis** devem ser escritas em **lower camelCase**.
## Labels
- Noma das **labels** devem ser escritas em **lower camelCase**.
